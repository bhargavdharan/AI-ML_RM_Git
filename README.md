# AI-ML_RM_Git
A Practical Git Learning Repository

## Overview
This repository is dedicated to learning Git through hands-on practice.  
It serves as a sandbox where you can experiment with Git commands, branching, merging, conflict resolution, version tracking, and working with GitHub.

The purpose is to help build strong version-control fundamentals essential for software development, AI/ML projects, DevOps workflows, and collaborative programming.

---

## Objectives
This repository helps you:

- Understand Git at a practical, beginner-friendly level  
- Learn the complete Git workflow (local and remote)  
- Gain confidence with commits, branches, merges, and pull requests  
- Use GitHub effectively for version tracking  
- Improve your commit habits with clean messages  
- Maintain organized development workflows for larger AI/ML projects  

---

## Repository Structure

You can organize your practice content like this:

```
AI-ML_RM_Git/
```

## Getting Started

### Clone This Repository

Using HTTPS:

```bash
git clone https://github.com/bhargavdharan/AI-ML_RM_Git.git
cd AI-ML_RM_Git
```

Using SSH:

```bash
git clone git@github.com:bhargavdharan/AI-ML_RM_Git.git
```

Initial Git Setup (Required Only Once)

Configure your Git identity:

```
git config --global user.name "Dharan"
git config --global user.email "your_github_email@example.com"
```

Verify your settings:

```
git config --global --list
```
